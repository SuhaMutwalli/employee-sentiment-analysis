# Employee Sentiment Analysis
# Author: Suha Mutwalli

This repository contains my final assessment submission for the Springer Capital AI Internship.

## Files Included
- employee_sentiment_analysis.ipynb — Main Jupyter Notebook
- test.csv — Input dataset
- test_labeled_vader.csv — Output dataset with VADER sentiment labels

## How to Run
1. Install requirements:
   pip install pandas numpy nltk scikit-learn
2. Open the notebook in Jupyter:
   jupyter notebook employee_sentiment_analysis.ipynb
3. Run all cells to reproduce the sentiment analysis.

## Summary
The notebook performs basic sentiment analysis on employee comments using VADER. It includes preprocessing, sentiment scoring, and generating labeled output.
